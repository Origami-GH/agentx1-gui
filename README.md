# agentx1-gui
受到<https://github.com/mayswind/luci-app-njitclient>的启发，写了一个简单的agentx1 LuCI界面，勉强能用，可能会有一些问题

很快会提供makefile以供编译成ipk

agentx1本体及其使用说明：<https://bitbucket.org/CrazyBoyFeng/agentx1>

### 参数对照
| agentx1 | agentx1-gui |
|:---:|:---:|
| -L | LAN |
| -W | WAN |
| -p | Promiscuous Mode |
| -a | DHCP |
| -i | 静态地址 |
| -n | 子网掩码 |
| -g | 网关 |
| -d | DNS |
| -u | 用户名 |
